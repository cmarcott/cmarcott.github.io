Professional Job Experience and Skills
Software Developer

NCR Corporation, Waterloo

At NCR I worked primarily as a back-end developer creating API's for bank transaction software. Much of my work involved working in Java on our stack of microservices which were deployed to a Docker/Kubernetes environment.

A typical day for me consisted of writing new features for our microservices, creating unit and integration tests for these features using JUnit and Cucumber/Gherkin, and ensuring these features would properly be built into Docker images and deploy to our Kubernetes cluster utilizing our Jenkins automated build server software within our continuous integration (CI) environment.

Additional duties would include maintaining Kubernetes clusters and debugging issues at a sysadmin level within development environments or developing our AngularJS POC application in order to showcase new features. All this would be done whilst attending standard agile development meetings such as daily standup or sprint planning.

IT Developer
